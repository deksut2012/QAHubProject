UX Rules

  • Fail Result ต้องบังคับกรอก Actual Result และ Evidence
  • ทุกหน้าต้อง Filter ตาม Product, Module, Status และ Owner ได้
  • รายการจำนวนมากต้องรองรับ Pagination และ Saved View
  • สถานะสำคัญต้องใช้สีสม่ำเสมอทุกหน้า
  • หน้าจอ Execution ต้องใช้งานบน Laptop ได้สะดวกที่สุด
  • AI Output ต้องเป็น Draft และผ่าน Human Review ก่อน Publish
  • แสดง Confidence, Assumption และ Missing Information ทุกครั้ง
  • Mask Password, Token และ PII ก่อนส่งข้อมูลให้ AI
