## Definition of Done

- Discovery workshop ครบผู้แทน PO, QA, Dev, DevOps และ Security
- Current workflow และ pain point มีหลักฐานอ้างอิง
- เลือก Pilot Product และระบุเหตุผล
- MVP scope และ out-of-scope ได้รับอนุมัติ
- KPI มี baseline, target, data source และ owner
- Integration/Security constraints ได้รับการยืนยัน
- Initial backlog ถูกจัดลำดับ
- Architecture direction และ owner ได้รับอนุมัติ
